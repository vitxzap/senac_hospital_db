# Functions
Funções ajudam automatizar blocos de códigos que se repetem mais de uma vez durante o desenvolviemento do projeto.


## Functions disponíveis
| Function | Sobre |
|--|--|
| calcular_idade | Calcula a idade de um paciente |
| dias_internacao | Calcula quantos dias o paciente ficou internado |
| medico_disponivel | Verifica se determinado médico está disponível |
